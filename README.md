# ProtoParser

This Julia module serves two purposes:
- Being a collection of protobuf Schemas used by QPerfect
- Provides all neccessary functions to parse and save a protobuf file to/from a MIMIQ Ciruit

The source code can be found in the src folder and is separated in three subfolder:
- proto: Contains all the protobuf schemas
- julia: Contains all the Julia code generated from the protobuf schemas
- parser: Contains all the Julia functions used to parse/save a file to/fom a MIMIQ Circuit
